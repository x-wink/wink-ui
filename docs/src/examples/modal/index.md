## 附加内容

### ModalCloseReason `type`

弹窗关闭原因联合类型，可选值：

<ul>
    <li v-for="(item, index) in modalCloseReasons" :key="index">{{ item }}</li>
</ul>

### modalCloseReasons `const`

弹窗关闭原因枚举列表，类型为 [`ModalCloseReason[]`](#modalclosereason-type)

import { modalCloseReasons } from 'wink-ui';
